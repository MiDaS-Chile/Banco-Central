Actividad II

Cree dos funciones para Œ≥(s,x), la primera utilizando la definci√≥n y la segunda utilizando su representaci√≥n en series de potencia. Las funciones no se deben ejecutar si el argumento s es negativo y un mensaje debe ser entregado al usuario.

gam1<- function(s,x){
  if(s < 0) stop("s debe ser positivo")
  myf<-function(t){
    t^(s-1)*exp(-t)
  }
  integrate(myf,0,x)$value
}

gam2<- function(s,x){
  if(s < 0) stop("s debe ser positivo")
  k<-0
  aux<-0
  while (!is.infinite(gamma(s+k+1))){
    aux=sum(x^(0:k)/gamma(s+0:k+1))
    k<-k+1
  }
  ifelse(is.infinite(aux),gamma(s),x^s*gamma(s)*exp(-x)*aux)
}
